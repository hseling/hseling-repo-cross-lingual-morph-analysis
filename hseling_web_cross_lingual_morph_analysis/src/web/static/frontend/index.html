About Us

Welcome from the cross-lingual-morph-analysis Team! We are 3 (previously 5) students from NRU-HSE, our names are Lorenzo Tosi, Vladimir Vlasov and Anastasia Khorosheva.
The algorithm we developed is able to produce lemma, part-of-speech tag and morphosyntactic annotations for words in a given language, based on annotated data of languages coming from the same family (e.g. Sardinian and Italian).
The full algorithm is explained in the paper we produced for the VarDial 2019 competition (https://www.aclweb.org/anthology/W19-1415.pdf).
